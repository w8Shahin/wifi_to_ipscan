# ============================================================================
# Python 3.6 or higher required
# Tested on: Python 3.6, 3.7, 3.8, 3.9, 3.10, 3.11, 3.12

# ============================================================================
# BUILT-IN MODULES (NO INSTALLATION NEEDED)
# ============================================================================
# All dependencies are built-in Python modules - No pip install required!
#
# The following modules are used (all included with Python):
# - socket          : Network communication
# - sys             : System-specific parameters
# - ipaddress       : IP address manipulation
# - datetime        : Date and time handling
# - concurrent      : Multi-threading support
# - threading       : Thread management
# - time            : Time-related functions
# - subprocess      : System command execution
# - re              : Regular expressions
# - platform        : Platform identification
# - os              : Operating system interface
# - itertools       : Iterator functions
# - urllib          : URL handling (for public IP)

# ============================================================================
# SYSTEM TOOLS (Platform-specific)
# ============================================================================
# These are system-level tools, installed via package manager:
#
# Windows:
#   - ipconfig      : Network configuration (built-in)
#   - arp           : ARP table access (built-in)
#   - route         : Routing table (built-in)
#   - nbtstat       : NetBIOS info (built-in)
#   - ping          : Network testing (built-in)
#
# Linux/Mac:
#   - ip            : Network configuration (install: iproute2)
#   - arp           : ARP table access (install: net-tools)
#   - route         : Routing table (install: net-tools)
#   - nmblookup     : NetBIOS lookup (install: samba-common-bin)
#   - ping          : Network testing (built-in)
#
# Termux (Android):
#   - ip            : Network configuration (pkg install iproute2)
#   - arp           : ARP table (pkg install net-tools)
#   - ping          : Network testing (built-in)
